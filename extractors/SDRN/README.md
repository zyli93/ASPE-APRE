# SDRN
Source code of the paper "Synchronous Double-channel Recurrent Network for Aspect-Opinion Pair Extraction, ACL 2020."

#### Requirement:

```
  python==3.6.8
  torch==0.4.0
  numpy==1.15.4
```

#### Dataset:
